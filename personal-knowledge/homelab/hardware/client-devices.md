# Usage

- **Mac Minis** are the primary desktop development machines. Fixed location, always on, connected to the home network.
- **MacBook Pro** for development on the move.
- **iPad Pros** for light coding via Telegram, remote access to the lab, and reading/reviewing code.

All client devices connect to the servers ([Intel](/homelab/hardware/intel.md) and [AMD](/homelab/hardware/amd.md)) over the local network for Docker builds, service access, and file sharing.
